CourierTask
===========
Курьер начинает работу со станции STARTING_STATION
Начало рабочего дня - STARTING_TIME
Длительность рабочего дня - не менее чем WORKING_DAY_DURATION

После того, как сгенерировался рабочий день, появляется меню:

Что вы хотите сделать дальше?
1. Вывести полную информацию о поездках курьера за сегодня
2. Вывести количество и стоимость поездок за сегодня
3. Узнать информацию по конкретной станции
4. Выйти из программы
